# llm
   - 大模型
   

# token
   - 大模型处理文本的「最基本单位」
   - 0.75个英文
   - 1-2个汉字

# context
   - 概念
     - 大模型每次处理任务时所接受的信息总和（大模型的临时记忆体）

# context window(上下文窗口)     
     - context 能容纳的最大「token数」

# prompt
     - 用户或者系统当前给大模型下达的具体指令或问题（user prompt or system prompt）

# rag

# tool
  - 让大模型能够感知外部环境的函数

# mcp
  - 工具接入平台的统一规范 model context protocol (统一的工具接入标准协议)


# agent
   - 自主规划和调用工具直到用户解决问题的程序
